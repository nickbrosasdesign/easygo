EasyGo Design System

Component library and design tokens for EasyGo — an accessible routing app for people with mobility and cognitive disabilities.

Figma: EasyGo Redesign 2026 Lexend Typeface WCAG 2.1 AA

Overview

Brand personality & design principles

EasyGo's visual identity is built on clarity, trust, and empowerment. Every design decision prioritises low cognitive load — interactions are predictable, labels are explicit, and colour is used purposefully rather than decoratively.

The palette is anchored by a confident Navy Blue (#005299) that communicates reliability and authority, paired with a crisp white ground. A 4px grid governs all spacing. Lexend — chosen for its proven readability improvements for users with dyslexia — is the sole typeface used throughout the system.

Corner radii are generous (up to 32px) to give the UI a friendly, approachable character while remaining professional. Interactions always surface focus states to support keyboard and switch-access navigation.

Text Inputs

input/text & input/text-area

Text inputs collect user input. States: Primary, Hover (blue/100 border glow), Focus (blue/400 border + blue/100 ring), Disabled (grey/100 background). Text area provides multi-line entry with the same visual language.

Dropdown selector for choosing from a predefined list. States: Closed/Open × Primary/Hover/Focus/Disabled. 48px height. Options inherit Lexend, 16px, base/black on white background.

Pagination

pagination

Navigation control for paginated content. Includes prev/next arrow controls and numbered page buttons. Current page uses primary blue fill. Disabled state applies to arrows at the first or last page.

Navigation

nav

The primary navigation menu. Five main app sections — each as a large, touch-friendly nav item with a 64px circular icon and a 24px bold label. Designed for maximum legibility and minimal cognitive overhead.
